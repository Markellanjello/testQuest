Функционал строится на вводе команды в терминал
1. show - вывод всех данных таблицы в формате dataframe
2. edit - редактирование записи в формате как взял так и положил
3. delete - удаление записи
4. search - поиск записи по значению или значениям, в формате введения запросов через запятую
5. dump - отправка записи в установленной форме
6. exit - возврат на страницу назад


Я знаю что не очень оптимально использовать подобный формат работы с базой. Но вроде как все требования выполнены.
